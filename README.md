# SPIN website rewrite
## What is SPIN
SPIN is a information system for accidents and intervetions reports. It's used by Uprava RS za zaščito in reševanje for reporting all accidents in Slovenia.
## What is this project?
This React project is a better visualization tool for current [SPIN](https://spin3.sos112.si/javno/pregled) site.
## Reasons for this project
- No dark theme
- Mobile page is quite laggy
- Map is not working OK on mobile
